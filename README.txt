Les programmes Jeu de la vie 2.py et Jeu de la vie 3.py fonctionnent.

Les fichier .txt servent à charger les maps préconfigurées (non indispensables) et sont à mettre dans le même dossier que Jeu de la vie 2.py


La version 3 du programme est plus optimisé mais requiert d'installer numpy et scipy en exécutant les commandes suivantes dans un cmd (sur windows) ou un terminal (sous macOS) :
pip3 install numpy
pip3 install scipy
