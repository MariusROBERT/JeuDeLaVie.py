Les programmes Jeu de la vie 2.py, Jeu de la vie 3.py et Jeu de la vie 4.py fonctionnent.

Les fichier .txt servent à charger les maps préconfigurées (non indispensables) et sont à mettre dans le même dossier que Jeu de la vie 2.py


La version 3 du programme est plus optimisé mais requiert d'installer numpy et scipy en exécutant les commandes suivantes dans un cmd (sur windows) ou un terminal (sous macOS) :
pip3 install numpy
pip3 install scipy

La version 4 du programme est encore plus optimisé que la version 3 et ne requiert aucun package de plus que cette dernière.
Cependant, si vous n'avez pas installé numpy et scipy (indispensable pour la version 3), vous devez les installer pour la version 4.
